Topic 1

Procedural Fundamentals

In this topic you will learn about the basic programming constructs: iteration; branching and sequence. You will also learn about the data types to store your variables: boolean, integer, string, float point and char. This is the topic that will help you to gain an understanding on programming fundamentals.

Topic 2

Binary and Hex Numbers

In this topic you will learn how to convert numbers from denary to binary and hexadecimal. You will also learn hot to write different versions of binary Two's complement and sign and magnitude, which will allow negative numbers to represented.

Topic 3

Types of Networks

In this topic you will learn about different types of networks. You will also learn about different network topologies which is how a network is laid out and how the devices are arranged. You will also learn in this topic the difference between a standard and a protocol.

Topic 4

Devices and Storage

In this topic you will learn what an input and output device is and what the difference between them is. You will also learn about the different types of storage devices like optical, magnetic and solid state. It will also tell you the different standards that there are for the different types of storage.
